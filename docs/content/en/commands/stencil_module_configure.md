## stencil module configure

```bash
NAME:
   stencil module configure

USAGE:
   stencil module configure [command options] configure module

DESCRIPTION:
   updates a module with the provided name in the current directory

OPTIONS:
   --remove-native-extension  Removes native extension configuration for the provided module (default: false)
   --help, -h                 show help

```
